# OwlToUml
Converter from OWL (OWL-API compatible) to UML (Eclipse UML2). Supports export to Enterprise Architect native UML.

# Example Usage
```
java -jar OwlToUml-1.0.0.jar --ontology http://www.w3.org/ns/prov-o --prefix prov
```
